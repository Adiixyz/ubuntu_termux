# ubuntu-termux
Install Ubuntu in Termux

## Installing
```cmd
$ apt install git
$ apt install bash
$ git clone https://github.com/Adiixyz/ubuntu_termux
$ cd ubuntu_termux
$ bash install.sh
```

### install it without $ in [`#installing`](https://github.com/Adiixyz/ubuntu_termux#installing)

## Run...
```cmd
$ bash run.sh
```
# Run Simply
Run `simple.sh` first
- Once youre done. type `ubuntu`
